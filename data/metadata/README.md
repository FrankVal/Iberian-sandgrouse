# metadata

Metadata describing the datasets used in this repository, including variable definitions, units, and lookup tables.

Contents:
- codebook_sdm_occurrence.csv: SDM presence/absence and variable definitions.
- codebook_demography_fpca_glmm.csv: demography and class-metric variable definitions (with units).
- sites_lookup.csv: stable lookup of unique 10×10 km UTM cells used in the analyses.
